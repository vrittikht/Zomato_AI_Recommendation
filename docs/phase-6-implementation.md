# Phase 6 Implementation (Response Presentation Layer)

This layer focuses on transforming the structured LLM output into human-readable and visually appealing formats.

## Implemented in `src/response_presentation/`

- `cli_renderer.py`: Colorized terminal output using ANSI codes.
- `html_renderer.py`: Generates a high-end, responsive HTML report with:
    - Glassmorphism design
    - Interactive hover effects
    - Smooth entry animations
    - Premium typography (Outfit by Google Fonts)
- `service.py`: Orchestrates the rendering process.

## Outputs

- Console output (immediate)
- `docs/recommendation_report.html`: A standalone visual report.

## How to Run

```bash
python scripts/run_phase6.py
```

## Features

- **Rich Aesthetics**: Uses a dark-mode theme with vibrant accents (Sunset Pink / Cyan).
- **Responsive design**: Works on mobile and desktop.
- **Traceability**: Clearly indicates if the results were generated by LLM or falling back to deterministic ranking.
- **Micro-animations**: Subtle transitions for cards to enhance user experience.
